Custom Levels for GravityShift

Making custom levels for GravityShift is incredibly easy thanks to the custom level importer included in the Windows build of the game.


The Windows build can be downloaded here.


Step 1: Making the level

In order to create a level for GravityShift, you'll need to create a .txt file. Once you have done so, you can begin writing your level using a program such as notepad.


There are 2 main things to remember when making levels:


  1. The width and height of the text must be the same. For example, all lines must be the same length of characters, and the number of lines in the .txt file must be equal to the length of the lines.
  2. You should only use the characters listed below, the characters are case sensitive.

Step 2: Using the correct characters

The usable charcters as well as their purpose are listed in the table below:

Character Purpose
0 This will leave that area of the level blank. Strictly speaking this can be any character not listed below, but it is recommended to use 0 as it will always be supported.
X This will spawn an obstacle block which will block the player as they move.
P This will spawn a player block, multiple players in one level is allowed, but if one triggers a finish block, level will end.
F This will spawn a finish block, once again, multiple finish blocks are allowed, however, if one is triggered, the level will end.
L This will spawn a lock block, once again, multiple lock blocks are allowed, one key will unlock all in the level.
K This will spawn a key block, this is the key which is used to despawn lock blocks. Multiple are allowed, but they cannot be linked to certain lock blocks, and once one is triggered, all keys will despawn.

Stage 3: Importing the level

Now, in order to play your level, you must located the GravityShift.exe file. From there, go into "GravShift_Data\StreamingAssets\CustomLevels", and copy your .txt file into the folder.


Don't worrying about naming it in a special way, as long as the file end in .txt, and you have followed rule number 1, you should now see your level in the custom level menu of GravityShift, just click on it to play.



Just as a side-note, you are able to create levels with over 1000 blocks on each axis, the game will scale the blocks once they are placed. If your level does not show up on the "Custom Levels" screen, ensure its dimensions are the same, and make sure there is not an empty line at the end, the parser does not remove the new line charcter from the last line.
